# 概述 — 版本控制

所有 Spiral 组件都遵循 https://semver.org/，安全版本更新允许向后兼容。

## 当前版本

当前的框架版本是 `3.*`。之前的版本可以在[这里](https://github.com/spiral/framework/releases)找到。

| 版本     | PHP   | 发布日期     | 功能开发结束  |
|-------------|-------|--------------|----------------------------|
| 0.*        | ≥ 5.0 | 2013年11月   | 2017年12月                   |
| 1.*        | ≥ 7.2 | 2017年4月   | 2019年12月                   |
| 2.*        | ≥ 7.4 | 2019年5月   | 2022年12月                   |
| <b>3.*</b> | ≥ 8.1 | 2022年9月   | -                          |
